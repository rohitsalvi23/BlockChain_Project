# BlockChain_Project
x21127336_BlockChain_Project

Clone this repository:

    https://github.com/rohitsalvi23/BlockChain_Project
    
Install the following dependencies:
Web3:
    npm install web3

Bignumber:
    npm install bignumber

Ethereumjs-tx
    npm install ethereumjs-tx

Dotenv:
    npm install dotenv

Run distribute.js:

    node distribute.js
